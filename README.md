# KindlePi

A custom offline e-ink e-reader built on Raspberry Pi 5 with local AI (llama.cpp + Qwen2.5-1.5B).

## Hardware
- Raspberry Pi 5 8GB
- Waveshare 9.7" e-ink display (1200x825, IT8951)
- Waveshare UPS HAT (E) with 4x Samsung 21700 cells

## Features
- Reads epub files
- Renders text to e-ink display
- Local AI for word definitions and chapter summaries
- Fully offline, no cloud dependency
- GPIO button navigation
